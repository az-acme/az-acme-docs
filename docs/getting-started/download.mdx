<TabItem value="browser" label="macOS">

With homebrew installation is simple:

```bash
brew install az-acme/tap/cli
```

Once installed you can verify the installation by running the ```az-acme``` command with the ```--version``` flag to see the installed version.

```bash
az-acme --version
```
